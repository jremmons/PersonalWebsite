Realistic Image Simulation
Jason W. Trobaugh, R. Martin Arthur

We employ a linear systems model for the RF imaging system and a discrete-scatterer model for the acoustic characteristics of the surface. This physical model permits the simulation of images, three of which are shown here for the image plane at right. An actual image is shown for approximately the same image plane.
